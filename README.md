# cornhone-useful

Minimal webapp to handle the score during a friendly cornhole game.

## Overview

This project is a lightweight HTML-based scoreboard for casual cornhole matches.

## Features

- Track team scores during play
- Simple, minimal UI
- Static web app (no backend required)

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/hsk-dk/cornhone-useful.git
   ```

2. Open the project folder.
3. Open the main HTML file in your browser.

## Usage

- Launch the app in a browser.
- Update scores as each round is played.
- Reset when starting a new game.

## Tech Stack

- HTML (100%)
